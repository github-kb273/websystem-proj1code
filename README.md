CS540 project 1 code for websystem word game<BR>
Requirements:  Python 3.11 or above<BR>
Packages:  <BR>
Flask<BR>
requests<BR>
<BR>
If you want to run this game in WinOS, initialize by:<BR>
python -m venv env0<BR>
env0\Scripts\activate<BR>
pip install Flask<BR>
pip install requests<BR>
<BR>
Run:<BR>
change directory to repository directory<BR>
type ".\wordfun.py" to start server<BR>
<BR>
Browser:<BR>
Go to address:  http://127.0.0.1:5000<BR>
