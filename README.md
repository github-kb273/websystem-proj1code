CS540 project 1 code for websystem word game Requirements: Python 3.11 or above

Packages:

Flask 

requests



If you want to run this game in WinOS, initialize by: 

python -m venv env0

env0\\Scripts\\activate

pip install Flask

pip install requests



Run: 

change directory to repository directory 

type ".\\wordfun.py" to start server



Browser: 

Go to address: http://127.0.0.1:5000
